# SaaS | Translation Chat

![SaaS](https://github.com/MarceloMafraBJJ/saas-translation/assets/84472778/d7ede730-8789-4fa2-9827-f5bb023e12b1)


Features:

- Nextjs
- TailwindCSS
- NextAuth
- Firebase
- Stripe

### Install packages

```shell
npm i
```

### Setup .env file

```js
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
FIREBASE_PROJECT_ID=
FIREBASE_CLIENT_EMAIL=
FIREBASE_PRIVATE_KEY=
STRIPE_SECRET_KEY=
```

### Run

```shell
npm run dev
```
